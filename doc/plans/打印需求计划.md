# 打印需求计划

## 需求描述

### 功能1：插入 console.log

选中文件中的部分内容（支持多选），按下快捷键 `Ctrl+Alt+L` 后，在下一行添加 `console.log` 语句。

### 功能2：批量管理 console.log

- `Ctrl+Alt+D` - 删除文件中与配置格式一致的 console.log
- `Ctrl+Alt+C` - 注释文件中与配置格式一致的 console.log
- `Ctrl+Alt+U` - 取消注释文件中与配置格式一致的 console.log

**重要**：仅处理与配置的 `consoleLogTemplate` 格式一致的 console.log，不一致的不进行处理。

### 示例

**原始代码：**
```javascript
const obj = { name: '张三' }
```

**选中内容：** `obj`

**默认模板 `${value}, ${name}` 效果：**
```javascript
const obj = { name: '张三' }
console.log(obj, `obj`)
```

**自定义模板 `'前置', ${value}, '后置'` 效果：**
```javascript
const obj = { name: '张三' }
console.log('前置', obj, '后置')
```

**自定义模板 `${value}, ${name:变量名:}` 效果：**
```javascript
const obj = { name: '张三' }
console.log(obj, `变量名:obj`)
```

### 功能2示例

**原始代码：**
```javascript
const obj = { name: '张三' }
console.log(obj, `obj`)  // 匹配模板
console.log('其他', x)   // 不匹配模板
console.log(obj, `obj`)  // 匹配模板
```

**执行删除后：**
```javascript
const obj = { name: '张三' }
console.log('其他', x)   // 不匹配，保留
```

**执行注释后：**
```javascript
const obj = { name: '张三' }
// console.log(obj, `obj`)  // 匹配模板，已注释
console.log('其他', x)       // 不匹配，保留
// console.log(obj, `obj`)  // 匹配模板，已注释
```

## 技术分析

### 1. 功能1：插入 console.log

#### 功能实现要点

- **命令注册**：需要在 `package.json` 中注册新命令 `variableTranslator.insertConsoleLog`
- **快捷键绑定**：绑定 `Ctrl+Alt+L`，仅在编辑器有选中文本时可用
- **核心逻辑**：
  1. 获取选中的文本内容
  2. 读取模板配置
  3. 解析模板变量（`${value}` 和 `${name}`）
  4. **使用 AST 解析确定正确的插入位置**
  5. 在正确位置插入 `console.log(解析后的参数)`
  6. 处理缩进（保持与当前行相同的缩进）
  7. 处理边界情况（如已是最后一行）

#### 插入位置计算（参考 turbo-console-log）

使用 `acorn` 库解析代码生成 AST，根据选中变量的上下文确定插入位置：

| 代码类型 | 插入位置 |
|----------|----------|
| 普通赋值 `let x = 1` | 下一行 |
| 对象字面量 `let x = {...}` | 对象结束后下一行 |
| 数组赋值 `let x = [...]` | 数组结束后下一行 |
| 函数赋值 `let x = function() {...}` | 函数结束后下一行 |
| 函数调用 `let x = foo()` | 下一行 |
| 三元表达式 `let x = a ? b : c` | 下一行 |
| 属性访问 `let x = obj.prop` | 下一行 |

### 2. 功能2：批量管理 console.log

#### 功能实现要点

- **命令注册**：注册 `deleteConsoleLog`、`commentConsoleLog`、`uncommentConsoleLog` 三个命令
- **快捷键绑定**：
  - `Ctrl+Alt+D`：删除 console.log
  - `Ctrl+Alt+C`：注释 console.log
  - `Ctrl+Alt+U`：取消注释 console.log
- **核心逻辑**：
  1. 读取整个文件内容
  2. 根据配置的模板格式，构建匹配正则表达式
  3. 遍历每一行，检查是否匹配模板格式
  4. 根据命令执行相应操作：
     - **删除**：移除匹配的行
     - **注释**：在行首添加 `//` 或 `/* */`
     - **取消注释**：移除行首的注释符号
  5. 应用修改到文件

#### 匹配规则

根据 `consoleLogTemplate` 配置构建正则表达式：

| 模板 | 匹配正则 |
|------|----------|
| `${value}, \`${name}\`` | `/console\.log\(.+,\s*`.+`\)$/` |
| `'前置', ${value}, '后置'` | `/console\.log\('前置',\s*.+,\s*'后置'\)$/` |
| `${value}` | `/console\.log\(.+\)$/` |

#### 处理示例

**原始代码：**
```javascript
const obj = { name: '张三' }
console.log(obj, `obj`)  // 匹配模板，处理
console.log('其他', x)   // 不匹配模板，跳过
console.log(obj, `obj`)  // 匹配模板，处理
```

**执行删除后：**
```javascript
const obj = { name: '张三' }
console.log('其他', x)   // 不匹配，保留
```

**执行注释后：**
```javascript
const obj = { name: '张三' }
// console.log(obj, `obj`)  // 匹配模板，注释
console.log('其他', x)       // 不匹配，保留
// console.log(obj, `obj`)  // 匹配模板，注释
```

**执行取消注释后：**
```javascript
const obj = { name: '张三' }
console.log(obj, `obj`)  // 取消注释
console.log('其他', x)   // 不匹配，保留
console.log(obj, `obj`)  // 取消注释
```

### 3. 实现方案

#### 方案一：简单实现（推荐）
- 在当前行下一行插入
- 自动复制当前行的缩进
- 插入格式：`console.log(selectedText, 'selectedText')`

#### 方案二：增强实现
- 支持配置输出格式（是否包含变量名字符串）
- 支持多行选择处理
- 支持批量插入

### 4. 代码位置

- **入口文件**：`src/extension.ts`
- **配置文件**：`package.json`
- **新增功能**：
  - 注册命令：`variableTranslator.insertConsoleLog`
  - 实现函数：`handleInsertConsoleLog()`
  - 注册命令：`variableTranslator.deleteConsoleLog`
  - 实现函数：`handleDeleteConsoleLog()`
  - 注册命令：`variableTranslator.commentConsoleLog`
  - 实现函数：`handleCommentConsoleLog()`
  - 注册命令：`variableTranslator.uncommentConsoleLog`
  - 实现函数：`handleUncommentConsoleLog()`

### 5. 注意事项

- **缩进处理**：需要读取当前行的缩进，并应用到新插入的行
- **边界情况**：
  - 文件末尾插入
  - 选中文本包含特殊字符
  - 多行选择（可选支持）
- **性能考虑**：轻量级操作，无需异步处理
- **匹配准确性**：仅处理与配置模板格式一致的 console.log
- **注释格式**：支持 `//` 和 `/* */` 两种注释格式
- **批量操作**：支持全文件批量处理

在 `package.json` 的 `contributes.configuration` 中添加配置项：

```json
{
  "contributes": {
    "commands": [
      {
        "command": "variableTranslator.insertConsoleLog",
        "title": "插入 console.log"
      },
      {
        "command": "variableTranslator.deleteConsoleLog",
        "title": "删除 console.log"
      },
      {
        "command": "variableTranslator.commentConsoleLog",
        "title": "注释 console.log"
      },
      {
        "command": "variableTranslator.uncommentConsoleLog",
        "title": "取消注释 console.log"
      }
    ],
    "keybindings": [
      {
        "command": "variableTranslator.insertConsoleLog",
        "key": "ctrl+alt+l",
        "when": "editorTextFocus && editorHasSelection && editorLangId =~ /^javascript|typescript|vue|jsx|tsx$/"
      },
      {
        "command": "variableTranslator.deleteConsoleLog",
        "key": "ctrl+alt+d",
        "when": "editorTextFocus && editorLangId =~ /^javascript|typescript|vue|jsx|tsx$/"
      },
      {
        "command": "variableTranslator.commentConsoleLog",
        "key": "ctrl+alt+c",
        "when": "editorTextFocus && editorLangId =~ /^javascript|typescript|vue|jsx|tsx$/"
      },
      {
        "command": "variableTranslator.uncommentConsoleLog",
        "key": "ctrl+alt+u",
        "when": "editorTextFocus && editorLangId =~ /^javascript|typescript|vue|jsx|tsx$/"
      }
    ],
    "configuration": {
      "title": "变量翻译助手",
      "properties": {
        "variableTranslator.enableConsoleLog": {
          "type": "boolean",
          "default": false,
          "description": "是否启用: 选中文本后按 Ctrl+Alt+L 快速插入 console.log"
        },
        "variableTranslator.consoleLogTemplate": {
          "type": "string",
          "default": "${value}, ${name}",
          "description": "console.log 参数模板。可用变量: ${value}=选中文本, ${name}=反引号字符串形式"
        }
      }
    }
  }
}
```

### 配置项说明

| 配置键 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `enableConsoleLog` | boolean | `false` | 控制是否启用 console.log 快速插入功能 |
| `consoleLogTemplate` | string | `` `${value}, \`${name}\`` `` | console.log 参数模板 |

### 模板变量语法

```
${variable}
${variable:prefix,suffix}
```

| 语法 | 说明 | 示例 | 结果 |
|------|------|------|------|
| `${value}` | 原始文本 | `obj` | `obj` |
| `${name}` | 原始文本（不添加引号） | `obj` | `obj` |
| `${name:prefix,suffix}` | 前缀+原始文本+后缀 | `obj` | `prefixobjsuffix` |

### 模板示例

| 模板 | 选中 `obj` 时输出 |
|------|-------------------|
| `` `${value}, \`${name}\` `` | `console.log(obj, \`obj\`)` (反引号) |
| `${value}, '${name}'` | `console.log(obj, 'obj')` (单引号) |
| `${value}, "${name}"` | `console.log(obj, "obj")` (双引号) |
| `${value}` | `console.log(obj)` |
| `'调试:', ${value}, \`${name}\`` | `console.log('调试:', obj, \`obj\`)` |
| `${value}, ${name:前,后}` | `console.log(obj, 前obj后)` |

### 配置检查逻辑

在 `handleInsertConsoleLog()` 函数中：
1. 首先检查配置 `ConfigManager.isConsoleLogEnabled()`
2. 若禁用，直接返回不执行任何操作
3. 读取模板配置
4. 解析模板变量：
   - `${value}` → 原始文本
   - `${name}` → `` `原始文本` ``
   - `${name:prefix}` → `` `prefix原始文本` ``
   - `${name:suffix}` → `` `原始文本suffix` ``
   - `${name:prefix,suffix}` → `` `prefix原始文本suffix` ``
5. 生成 console.log 参数并插入

## 预期效果

### 功能1：插入 console.log

用户选中任意变量或表达式后，按下 `Ctrl+Alt+L`，即可根据模板配置快速插入调试用的 `console.log` 语句。

#### 默认效果
```javascript
// 选中 obj 后按下快捷键
console.log(obj, `obj`)
```

#### 多选效果
```javascript
// 原始代码
let dsds = {
  name: '张三',
}
let dsds1 = {
  name: '张三',
}

// 同时选中 dsds 和 dsds1 后按下快捷键
let dsds = {
  name: '张三',
}
console.log(dsds, `dsds`)
let dsds1 = {
  name: '张三',
}
console.log(dsds1, `dsds1`)
```

#### 对象字面量效果
```javascript
// 原始代码
let dsds = {
  name: '张三',
}

// 选中 dsds 后按下快捷键（插入到对象结束后）
let dsds = {
  name: '张三',
}
console.log(dsds, `dsds`)
```

#### 自定义模板效果
```javascript
// 模板: '前置', ${value}, '后置'
// 选中 obj 后按下快捷键
console.log('前置', obj, '后置')
```

#### 带变量名效果
```javascript
// 模板: ${value}, ${name:变量名:}
// 选中 obj 后按下快捷键
console.log(obj, `变量名:obj`)
```

### 功能2：批量管理 console.log

#### 删除 console.log（Ctrl+Alt+D）

```javascript
// 原始代码

console.log(obj, `obj`)  // 匹配模板，删除
console.log('其他', x)   // 不匹配，保留
console.log(obj, `obj`)  // 匹配模板，删除

// 执行删除后
```

#### 注释 console.log（Ctrl+Alt+C）

```javascript

// 执行取消注释后
```
